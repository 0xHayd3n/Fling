# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, assume good intent, and don't make this space hostile to anyone.

Maintainers reserve the right to remove or edit contributions and to ban contributors whose behavior is harmful to the community.

Report incidents to: **contact@haydenseymour.com**. Reports will be handled confidentially.
